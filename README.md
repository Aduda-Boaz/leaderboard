## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm
- git

### Setup

- open terminal or bash
- run
  - `git clone https://github.com/Aduda-Boaz/leaderboard.git`

### Install

- run
  - `cd leaderboard`
  - `npm install`
  - `npm run build`
  - `npm start` to run the server

### Deployment

- run
  - `npm run build`
